Qué es WebAssembly

WebAssembly es un lenguaje de programación frontend que fue anunciado por primera vez en 2015. En 2016 se realizó la primera demostración de este lenguaje en Firefox, Chromium, Google Chrome y Microsoft Edge ejecutando una conocida demo llamada «Unity’s Angry Bots». Este lenguaje de programación, abreviado como WASM, utiliza un formato de código binario portable, bytecode). Gracias a él consigue una ejecución íntegra en navegador de los scripts del lado del cliente. Se trata de un lenguaje de programación de bajo nivel, y aunque fue creado originalmente como destino de compilación de código C y C++, también es compatible con otros lenguajes como Go y Rust.
